JAVA Installation

1. Download the latest package (JDK25) for Linux | Mac


OR just simply use wget (for Linux)

			wget https://download.oracle.com/java/25/latest/jdk-25_linux-x64_bin.tar.gz

2. Place the file in your home directory or any directory you prefer (e.g., /home/riza/).

3. Unpack the tarball and set the environment variables as follows:

			tar zxvf jdk-25_linux-x64_bin.tar.gz
		
			nano ~/.bashrc

5. Add the following lines to the end of your ~/.bashrc file (edit the code according to the location of the file on your machine).

			export JAVA_HOME='/home/riza/jdk-25.0.1'
		
			export PATH=$JAVA_HOME/bin:$PATH

6. Save and close the file. Once saved, have the current session read the file. 

			source ~/.bashrc

7. Verify installation:

			java -version

You should see the version of Java installed.

java version "25.0.1" 2025-10-21 LTS
Java(TM) SE Runtime Environment (build 25.0.1+8-LTS-27)
Java HotSpot(TM) 64-Bit Server VM (build 25.0.1+8-LTS-27, mixed mode, sharing)

PICARD Installation

1. Download the latest jar file here or use: 

			wget https://github.com/broadinstitute/picard/releases/download/3.4.0/picard.jar

2. Place the file in your home directory or any directory you prefer (e.g., /home/riza/).

3. Set the environment variable:

			nano ~/.bashrc

4. Add the following lines to the end of your ~/.bashrc file (edit the code according to the location of the file on your machine)

			export PICARD='/home/riza/picard.jar'

5. Save and close the file. Once saved, have the current session read the file. 

			source ~/.bashrc

6. Verify installation:

			java -jar $PICARD

Successful installation will display the PicardCommandLine usage message.

GATK4 Installation

1. Download the latest file here or use: 

			wget https://github.com/broadinstitute/gatk/releases/download/4.6.2.0/gatk-4.6.2.0.zip

2. Place the file in your home directory or any directory you prefer (e.g., `/home/riza/`).

3. To use GATK, the full path to the gatk-package-4.6.2.0-local.jar needs to be specified in the command, in this case it looks like this:

			/home/riza/gatk-4.6.2.0/gatk-package-4.6.2.0-local.jar

4. Set the environment variable:

			nano ~/.bashrc

5. Add the following lines to the end of your ~/.bashrc file (edit the code according to the location of the file on your machine)

			export GATK='/home/riza/gatk-4.6.2.0/gatk-package-4.6.2.0-local.jar'

6. Save and close the file. Once saved, have the current session read the file.

			source ~/.bashrc

7. Verify installation:

			java -jar $GATK

Successful installation will display the GATKCommandLine usage message.
